This folder contains solution to some simple CFD problems in 1D and 2D. The inspiration is taken from the famous [12 Steps to Navier-Stokes](https://lorenabarba.com/blog/cfd-python-12-steps-to-navier-stokes/) created by [Dr. Lorena A. Barba](https://lorenabarba.com/). I have tried to solve the same equations that she has presented in her tutorials using PINNs and have compared the result with the traditional FDM method used in CFD.
